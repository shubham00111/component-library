# Button

A clickable element that triggers an action. Available in five variants (`primary`, `secondary`, `destructive`, `ghost`, `outline`) and three sizes (`sm`, `md`, `lg`), implemented across three design languages.

All three variants share the same props interface — you can swap `<NeobrutalismButton>` for `<ShadcnButton>` without changing any props.
