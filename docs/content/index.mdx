## How It Works

**1. Browse** — Pick a component from the sidebar and see all three design languages side by side.

**2. Preview** — Click the live interactive preview to see states: hover, focus, disabled.

**3. Copy** — Hit "Copy" on the HTML or CSS tab. Paste it into any HTML file — no build tool, no npm install.
