# Toast

Fixed-position notification overlay. Variants: `success`, `error`, `warning`, `info`. Auto-dismiss via `timer` (ms).

> Toasts use `position: fixed` and appear at the corner of the viewport in production. The previews below are contained to the card for documentation purposes.
