A versatile action trigger available in three design languages, each with `primary`, `secondary`, `destructive`, `ghost`, and `outline` variants.

## Neobrutalism

Hard black shadow, thick border, saturated yellow primary. Translates on click.

<SnippetViewer react={snippets.neobrutalism.react} preview={<NeobrutalismButtonPreview />} />

## Shadcn-inspired

Neutral zinc palette, subtle shadow, 6 px radius, accessible focus ring.

<SnippetViewer react={snippets.shadcn.react} preview={<ShadcnButtonPreview />} />

## Flowbite-inspired

Blue `#1c64f2` primary, 8 px radius, smooth hover transition, blue glow on focus.

<SnippetViewer react={snippets.flowbite.react} preview={<FlowbiteButtonPreview />} />

## Glassmorphism

Frosted-glass panels with `backdrop-blur`, semi-transparent whites, glowing borders. Place inside a gradient container for the full effect.

<SnippetViewer react={snippets.glassmorphism.react} preview={<GlassmorphismButtonPreview />} />

## Material Design 3

Google's Material You system — purple `#6750A4` primary, tonal lavender container, pill-shaped `20px` radius, gentle elevation shadows.

<SnippetViewer react={snippets.md3.react} preview={<Md3ButtonPreview />} />

## Neumorphism

Soft UI — elements appear to extrude from the background via dual-tone shadows. Place inside a `#e0e5ec` container for the effect to work.
